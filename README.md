# 🎧 VibeSync

**See the vibe. Share the moment.**

VibeSync is a social music app inspired by AirBuds. Instead of just showing songs, it lets users share their **mood, energy, and culture** through music.  
Think of it as a **social vibe network** where music becomes a way to connect with friends and discover new vibes.

---

## 🚀 Features (MVP)

- 🔑 User Authentication (Email / Google)
- 🎵 Spotify integration (planned)
- 📰 Vibe Feed (scrollable posts with song, user, and mood)
- ❤️ Reactions: Like, Fire, Comment
- 🎭 Mood Tags: 🔥 Hype, 😌 Chill, 💔 Emotional
- 👤 User Profiles (future)

---

## 🛠️ Tech Stack

**Frontend:** React Native (Expo)  
**Backend:** Firebase (Auth + Firestore)  
**APIs:** Spotify API (planned)
